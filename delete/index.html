계정 삭제 안내

아스트라 도미니언 (Astra Dominion) — 계정 및 데이터 삭제

방법 1 — 게임 내에서 삭제 (권장)

게임 실행 → 설정 → 계정 → 계정 삭제 메뉴에서 직접 신청할 수 있습니다. 본인 확인이 즉시 이루어지므로 가장 빠른 방법입니다.

방법 2 — 웹으로 삭제 요청

앱을 삭제했거나 게임에 접속할 수 없는 경우, 아래 양식으로 삭제를 요청할 수 있습니다. 접수 후 계정 확인을 거쳐 처리되며, 결과를 이메일로 안내합니다.

계정 삭제 요청 양식 열기

양식 입력 항목: 게임 닉네임, 계정 UID(설정 화면에서 확인 가능), 회신받을 이메일. 양식 이용이 어려우면 support@durugames.com으로 같은 정보를 보내주세요.

삭제 처리 절차

  1. 삭제 신청 시 7일의 유예 기간이 시작됩니다.
  2. 유예 기간 내 해당 계정으로 로그인하면 삭제 신청을 철회할 수 있습니다. 철회는 해당 계정으로 로그인할 수 있는 경우에만 가능합니다.
  3. 게스트 계정(소셜 미연동)은 삭제 신청 후 앱을 삭제하거나 기기를 변경하면 로그인 수단이 사라져 유예 기간 중이라도 철회할 수 없으며, 삭제는 예정대로 진행됩니다.
  4. 유예 기간이 지나면 계정과 아래 데이터가 영구 삭제되며 복구할 수 없습니다.

삭제되는 데이터

  • 계정 정보(UID, 닉네임, 소셜 연동 정보), 게임 진행 데이터(행성·건물·유닛·연구·업적)
  • 보유 재화(에테르 포함 — 유·무상 불문 소멸, 환불되지 않음)
  • 우편, 탐사 기록, 회사(길드) 소속 정보

보관되는 데이터

「전자상거래법」에 따라 결제·환불 관련 기록은 법정 기간(5년) 동안 분리 보관 후 파기됩니다. 자세한 내용은 개인정보처리방침을 참고하세요.

삭제 전 확인하세요. 미사용 유료 에테르가 있다면 삭제 전에 청약철회(구매 후 7일 이내) 절차를 먼저 진행하세요. 계정 삭제는 환불을 대체하지 않으며, 삭제 후에는 잔여 재화가 소멸됩니다.
